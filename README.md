ESMS: Escape Room Managment System
==================================

## Installation

### Prerequisites

* Docker for Mac

### Installation

1. Clone the repository

    ```bash
    git clone git@github.com:nogohr/escape-room.git && cd escape-room
    ```

2. Install

    ```bash
    make install
    ```

Usage
-----

1. Start services

    ```
    make up
    ```
